About This Site

I've been teaching myself the ins and outs of web design and development since I was 12. Redesigning and rebuilding my personal website is one of the ways I keep my HTML, CSS, and JavaScript skills sharp, and how I test the waters with new techniques. I believe personal websites are valuable learning tools for new designers and developers and one of my goals is to make an inspirational website for other aspiring web professionals. My website is built from scratch in the Brackets text editor and makes use of Font Awesome icons, the Ikooni Outline icon set by Laura Reen, and the free font Henry Morgan Hand by Manfred Klein.
